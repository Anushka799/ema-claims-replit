
# Ema Claims Prototype — Replit-ready project

This repository contains a small React prototype for the **Ema APM Take-Home Project** (Claims Processing).

## What it includes
- `src/App.jsx` — Main React component (AI FNOL Assistant + Auto Triage + Adjuster Workspace). Mocks AI functions.
- `src/main.jsx` — App bootstrap.
- `index.html` — Vite entry.
- `src/index.css` — Minimal styles.
- `package.json` — Vite + React scripts.
- `slides.md` — Slide content for copy/paste into Google Slides or PowerPoint.

## How to run (locally)
1. Install Node 18+.
2. `npm install`
3. `npm run dev`
4. Open `http://localhost:5173` (or the port Vite shows).

## How to run on Replit
1. Create a new Replit using the **Vite + React** template or **Create React App**.
2. Replace `src/App.jsx`, `src/main.jsx`, and `index.html` with files from this project.
3. Run — Replit will provide a public URL you can submit.
